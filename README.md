# Pokemon GO React

Pokemon Go React by Marcus Leonato

### Instalação

```
git clone https://github.com/marcusleonato/react-redux-pokemon.git
npm install
npm start
```

## Uso
 Após a instalação, abra o endereço http://localhost:3000/ em seu navegador.
